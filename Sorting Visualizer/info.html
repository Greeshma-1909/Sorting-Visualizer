Content of the page

We have learnt sorting algorithms like bubble sort, selection sort, quick sort ,insertion sort and so many sorting algorithms but we often fail to understand the core idea of a particular algorithm, may be we are unable to visualise how they work .

So the most easiest way to understand about these algorithms is visualization.

This is a fun web application that helps you to visualize sorting algorithms. By visualizing them, you can compare how the algorithms differ, and how well they work with different sized inputs.

Things to know

  • Enter the size of the array.
  • Enter the elements of the array.
  • Use the slider to change the animation speed.
  • Then click the submit button.
  • Then select the sorting algorithm.

During the animation, there are some colors to know:

  • Yellow indicates the index of the current iteration
  • Red indicates the indices that have swapped values during the iteration
  • Blue is the color which is default to all indices.
  • Green indicates the indices that have fixed their positions.
